## 💡 Key Takeaways

- Sets store unique values.
- Duplicate values are removed automatically.
- Sets are unordered.
- add() inserts a new element.
- remove() deletes an element and raises an error if it doesn't exist.
- discard() deletes an element without raising an error.
- union() combines two sets.
- intersection() finds common elements.
- difference() finds elements present in one set but not the other.
- Membership can be checked using the `in` operator.
